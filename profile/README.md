# Spot - 취미 공유 강사 플랫폼

### 취미로 연결되는 사람들
프로 강사가 아닌 **또래가 호스트**가 되어 취미를 나누는 마켓플레이스입니다.
<br>
"수원 화서동 기타 같이 쳐볼 분, 1:1 5천원, 저희 집 거실에서 해요" 같은 **소규모 모임**을 만들고 참여할 수 있습니다.
<br>
**합성 콘텐츠 파이프라인**이 가상에서 시뮬레이션하여, 참여를 유도합니다.
